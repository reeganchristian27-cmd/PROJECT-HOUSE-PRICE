# House Price Prediction

## Project Overview

This project predicts house prices using machine learning. It uses features like area, number of rooms, location, and other factors to estimate the price of a house.

## Dataset

House Prices - Advanced Regression Techniques (Kaggle)

## Steps Involved

1. Data Cleaning

   * Handled missing values
   * Removed unnecessary columns

2. Feature Engineering

   * Created new features
   * Converted categorical data to numerical

3. Feature Scaling

   * Applied StandardScaler

4. Model Training

   * Linear Regression
   * Ridge Regression
   * Lasso Regression
   * Gradient Boosting

5. Model Evaluation

   * Mean Absolute Error (MAE)
   * Mean Squared Error (MSE)
   * R2 Score

## Best Model

Gradient Boosting gave the best performance.
